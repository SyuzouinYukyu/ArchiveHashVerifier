# ArchiveHashVerifier_v1.0.5

## 概要

ArchiveHashVerifier_v1.0.5 は、ダウンロードファイルやアーカイブファイルの SHA-512 / SHA-256 / SHA3-512 / SHA3-256 ハッシュを検証する Windows 11 x64 向けの C# WinForms ツールです。

## 使い方

アプリを起動し、検証したい対象ファイル本体、`.sha512` / `.sha256` / `.sha3-512` / `.sha3-256` ハッシュファイル、またはフォルダを上段の領域へドラッグ＆ドロップします。検証は自動で開始され、完了後もアプリを閉じずに連続して検証できます。

## D&D対応

- 対象ファイル本体をD&Dしても検証可能
- ハッシュファイルをD&Dしても検証可能
- 複数ファイルD&Dに対応
- フォルダD&D時は配下を再帰的に走査
- 本体ファイルとハッシュファイルを同時にD&Dしても重複検証を回避

## 対応例

- `CK.zip` + `CK.zip.sha512`
- `CK.zip` + `CK.zip.sha256`
- `CK.zip` + `CK.zip.sha3-512`
- `CK.zip` + `CK.zip.sha3-256`
- `CK.zip` + `CK.sha512`
- `CK.zip` + `CK.sha256`
- `CK.zip` + `CK.sha3-512`
- `CK.zip` + `CK.sha3-256`
- `CK.mkv` + `CK.mkv.sha512`

ハッシュファイルは `HASH  filename`、`HASH *filename`、`filename: HASH`、`HASH` のみのように、行内に対象アルゴリズムの16進ハッシュ値が含まれる形式を検出します。

対象ファイル本体をD&Dした場合は、SHA3-512、SHA-512、SHA3-256、SHA-256 の順で対応するハッシュファイルを探します。SHA3 系は `.sha3-512` / `.sha3-256` を標準とし、`.sha3_512` / `.sha3_256` も検出します。

## 進捗表示

検証中はプログレスバーを表示します。複数ファイル検証時は全体進捗を表示し、現在処理中のファイル、処理済み件数、読み込み済み容量、読み込み速度 MB/s、ETA を表示します。

## 結果メッセージ

すべて OK の場合、以下の趣旨の正常終了メッセージを表示します。

「すべての対象ファイルでハッシュ値が一致しました。この検証範囲では、ファイルの破損・ダウンロード不完全・改ざんの形跡は認められませんでした。」

この表示はハッシュ値一致に基づくファイル完全性確認であり、絶対安全を保証するものではありません。NG / ERROR がある場合は、破損、ダウンロード不完全、改ざんの可能性、ハッシュファイル不足、対応する本体ファイル不足、読み取りエラー等を確認するための注意喚起メッセージを表示します。

## OK / NG / ERRORの意味

- OK: 期待ハッシュと実測ハッシュが一致
- NG: 期待ハッシュと実測ハッシュが不一致
- ERROR: 対応ファイルが見つからない、ハッシュ値を読めない、読み取り例外が発生した

OK / NG / ERROR 件数は int 型で管理し、サマリーには必ず数値として表示します。

## ログ保存方法

ログは自動保存しません。「ログを保存」ボタンをクリックした時だけ Windows の保存ダイアログを表示し、UTF-8 で任意の場所へ保存します。「ログをクリア」ボタンを押した場合のみログ欄を消去します。

## 安全性

検証処理は読み取り専用です。検証対象ファイルやハッシュファイルを削除、移動、変更しません。管理者権限や外部通信、外部 NuGet パッケージは不要です。

SHA3-256 / SHA3-512 の計算には .NET 8 以上の標準 `System.Security.Cryptography` API を使用します。外部パッケージは追加していません。

## 専用アイコン

専用アイコン `app.ico` を同梱しています。EXE、ウィンドウ左上、タスクバーにアイコンを適用しています。`app.ico` が存在しない、または読み込めない場合でもアプリは起動します。

## 既存 verify.bat / verify.ps1 との関係

既存の `verify.bat` / `verify.ps1` と同じ用途の GUI 版です。既存ファイルは削除せず、PowerShell 版とは独立して C# 単体でハッシュ検証します。
