# Changelog

## v1.0.5 - 2026-05-15

- `.sha3-256` 対応を追加
- `.sha3-512` 対応を追加
- SHA3-256 / SHA3-512 の検証処理を追加
- SHA3 系の `.sha3_256` / `.sha3_512` 表記ゆれ検出を追加
- 既存 SHA-256 / SHA-512 検証機能は維持
- アプリ名とEXE名を `ArchiveHashVerifier_v1.0.5` へ更新

## v1.0.4 - 2026-05-14

- 正常終了メッセージを、ハッシュ値一致に基づく検証結果であることが明確な文言へ修正
- NG時の注意喚起メッセージを修正
- ERROR時の注意喚起メッセージを修正
- アプリ名とEXE名を `ArchiveHashVerifier_v1.0.4` へ更新

## v1.0.3 - 2026-05-14

- 実行中プログレスバーを追加
- 全体進捗表示を追加
- 読み込み速度 MB/s 表示を追加
- ETA表示を追加
- 正常終了時の安心メッセージを追加
- NG / ERROR 時の注意喚起メッセージを追加
- 検証完了後も連続検証できる状態を維持

## v1.0.2 - 2026-05-14

- 連続検証できない問題を修正
- 検証完了後にD&D受付状態へ戻るよう修正
- 対象ファイル本体D&Dから .sha512 / .sha256 を自動検出する機能を追加
- ハッシュファイルD&Dから本体ファイルを自動検出する機能を改善
- D&D表示文言を分かりやすく変更
- OK / NG / ERROR 件数表示を安定化
- アプリ名を ArchiveHashVerifier_v1.0.2 へ統一

## v1.0.1 - 2026-05-14

- 起動処理を try-catch で保護し、起動時例外を MessageBox で表示
- MainForm のタイトルと表示設定を明示
- 起動時ログを追加
- app.ico 読込失敗時もアプリが起動継続するよう修正
- 単一ファイル publish 設定に `EnableCompressionInSingleFile` を追加

## v1.0.0 - 2026-05-14

- C# WinForms / .NET 8 による初版を追加
- SHA-512 / SHA-256 のハッシュ検証に対応
- 複数ファイルとフォルダのドラッグ＆ドロップ検証に対応
- ログ表示、ログ保存、ログクリア、終了ボタンを追加
- OK / NG / ERROR 件数を int 型で管理し、サマリーに数値表示
- アプリ専用アイコン `app.ico` を追加
- MainForm / EXE / タスクバーへアイコンを適用
